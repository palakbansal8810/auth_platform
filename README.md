# Auth Platform

Multi-provider authentication system with:
- Email & Password
- Google OAuth
- Okta SAML

## Tech Stack
- FastAPI
- PostgreSQL
- React
- Docker
- Nginx

## Local Setup

1. Clone repo
2. Create .env file
3. Run:
   docker-compose up --build

Frontend: http://localhost:3000  
Backend: http://localhost:8000/docs
